# 安裝 Flask via Terminal
:::info
1.安裝 **虛擬環境**
```=
python -m venv .venv
.\.venv\Scripts\activate.ps1
```
2.安裝 **Flask** 套件
```=
pip install Flask
```
:::

# 網頁, Javascript的控制
:::info
1. **`app.py`** 提供API串接跟資料存儲
2. **`index.html`** 控制畫面
:::

# call API
:::info
1. **`前端`** call 自己的後端api
2. **`後端`** call gemini的api或chatgpt的api
:::
